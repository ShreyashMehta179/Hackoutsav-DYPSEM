# Hackutsav 2026 – Official Hackathon Website

Hackutsav 2026 is a National Level 24-Hour Hackathon organized by D. Y. Patil School of Engineering & Management (DYPSEM). This website serves as the official platform for participants to explore event details, registration, prize pool, domains, organizers, judges, sponsors, and hackathon updates.

## Features
- Modern Cyberpunk UI Design
- Fully Responsive Layout
- Animated Intro Screen
- Live Countdown Timer
- Organizers & Judges Section
- Prize Pool Showcase
- Hackathon Domains
- Registration Portal
- Contact & Support Section

## Tech Stack
- React.js
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
